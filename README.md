## Setup

1. Open `script.js` and replace `YOUR_GEMINI_API_KEY_HERE` with your actual Gemini API key:
   ```javascript
   const API_KEY = 'your-actual-api-key-here';
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Visit `http://localhost:8000` in your browser

## How to Use

1. Describe exactly how you're feeling in the text area
2. Click "Generate Playlist"
3. Enjoy your personalized playlist with liner notes!
